> **- 이선생, SCM 현장에서 구른 15년의 경험을 가진 사람**

## 🎯 독자 진단 테스트: 당신의 SCM 레벨은?

**아래 질문들에 솔직하게 답해보세요:**

### 📋 SCM 실력 체크리스트

#### 🌱 **신입/주니어 (0-2년차)**
- [ ] SCM이 정확히 뭔지 아직도 헷갈린다
- [ ] 엑셀 말고 다른 시스템은 무서워서 못 건드린다  
- [ ] 상사가 시키는 일만 하고 있다
- [ ] "이게 왜 필요한 일인지" 모르겠다는 생각이 든다
- [ ] 실수할까봐 밤잠을 설친 적이 있다

#### 🚀 **중급자 (3-7년차)**
- [ ] MES, SAP, ERP 시스템을 어느 정도 다룰 수 있다
- [ ] 공급업체와 기본적인 협상은 할 수 있다
- [ ] 데이터 분석해서 보고서 만드는 것쯤은 식은 죽 먹기다
- [ ] 하지만 "다음 단계"가 뭔지 모르겠다
- [ ] 전문가라고 부르기엔 뭔가 부족한 기분이다

#### 🎯 **고급자 (7년+ 리더/컨설턴트)**
- [ ] 전사적 관점에서 SCM 전략을 세울 수 있다
- [ ] 위기 상황에서 팀을 이끌고 해결책을 찾는다
- [ ] 다른 부서 사람들이 SCM 관련해서 나에게 조언을 구한다
- [ ] 미래 SCM 트렌드를 연구하고 적용한다
- [ ] 후배들을 체계적으로 멘토링할 수 있다

## 📚 이 책 활용 가이드: 경력별 읽는 법

### 🌱 **신입/주니어라면** → "위로와 희망 찾기"
```
📖 추천 독서 순서:
프롤로그 → 1-3장 (현실적응 + 실수담) → 4-6장
```

**포커스**: 
- "나만 이런 실수 하나?" 싶을 때 위로 받기
- 기초 개념과 실무 적응 노하우 습득
- 선배의 리얼한 경험담으로 마음의 준비 하기

### 🚀 **중급자라면** → "성장 로드맵 찾기"
```
📖 추천 독서 순서:  
프롤로그 → 4-6장 (전문성 개발) → 7-8장 (위기관리)
```

**포커스**:
- 전문성을 한단계 업그레이드하는 방법
- 리더십과 크로스 펑셔널 협업 스킬
- 컨설턴트로의 커리어 전환 준비

### 🎯 **고급자/리더라면** → "통찰과 멘토링 노하우"
```
📖 추천 독서 순서:
프롤로그 → 6-8장 (리더십 + 위기관리) → 전체 복습
```

**포커스**:
- 전략적 사고와 조직 변화 관리
- 위기 상황에서의 리더십 발휘
- 후배 멘토링과 팀 빌딩 노하우

## 🤔 왜 이 책을 써야겠다고 생각했나?

### 📖 기존 SCM 책들의 한계

**대부분의 SCM 책들:**
- 이론 위주의 딱딱한 내용 📚
- 해외 사례 중심 (한국 현실과 안 맞음) 🌍  
- 성공 사례만 나열 (실패담은 없음) ✨
- 대기업 임원급 관점 (현장 실무자와 거리감) 👔

**내가 원했던 책:**
- **진짜 현장에서 겪은 이야기** 💀
- **실수하고 망하고 다시 일어선 과정** 🔄
- **주니어부터 시니어까지 단계별 성장** 📈
- **한국 기업 문화에 맞는 실전 노하우** 🇰🇷

### 💡 "없으면 내가 만든다"

그래서 결심했다. **15년간 내가 겪은 모든 것을 있는 그대로 써보자.**

- ✅ **햇빛 없는 지하 공장**에서의 절망
- ✅ **회사 망칠 뻔한 대참사** 실수담  
- ✅ **멘토들로부터 배운** 진짜 노하우
- ✅ **위기를 기회로 바꾼** 드라마틱한 순간들
- ✅ **연봉 2배 올린** 컨설턴트 전환기

**한 줄도 과장하지 않겠다. 다 실화다.**

## 👨‍💼 저자 소개: 이선생이 누구냐?

### 📅 **15년 SCM 여정 요약**

**2010-2011**: 대기업 공장 품질관리/생산관리 → 3개월 17일 만에 탈출 🏃‍♂️

**2011-2014**: 중견기업 외자구매 → SCM의 'S'자도 모르던 시절 💼

**2014-2017**: PI팀(프로세스 혁신) → 진짜 일하는 방법을 배우다 🎯

**2017-2020**: SCM 분석가 → 이론을 실무에 적용하는 재미 📊

**2020-2023**: SCM 팀장 → 위기관리와 리더십의 실전 ⚡

**2023-현재**: 글로벌 SCM 컨설턴트 → 200억 프로젝트의 기적 🔥

### 🏆 **주요 성과들**

- **15년간 총 372억원 가치 창출**
- **예측 정확도 40% → 87%** 향상 달성
- **위기 상황 72시간** 만에 해결한 레전드
- **5억 손실을 10억 수익**으로 바꾼 프로젝트
- **연봉 2,500만원 → 2억 5천만원** (10배 증가)

### 💬 **내가 하고 싶은 말**

*"SCM은 교과서가 아니라 현실이다. 이론으로는 절대 배울 수 없는 것들이 있다. 실수하고, 망하고, 다시 일어서는 과정에서만 얻을 수 있는 진짜 실력이 있다."*

*"이 책은 나의 실패와 성공을 모두 담았다. 여러분이 나와 같은 실수를 반복하지 않길 바라는 마음으로 썼다. 그리고 혹시 지금 힘들어하고 있다면, 이것도 성장의 과정이라는 걸 알아주길 바란다."*
