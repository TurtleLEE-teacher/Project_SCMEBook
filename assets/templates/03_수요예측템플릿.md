# 수요 예측 템플릿 (Demand Forecasting Template)

## 📋 개요
체계적인 수요 예측을 위한 종합 분석 도구입니다. 다양한 예측 기법을 활용하여 정확한 수요 예측을 수행하고, 예측 정확도를 지속적으로 개선할 수 있습니다.

## 🎯 사용 목적
- 단기/중기/장기 수요 예측
- 다양한 예측 기법 비교 분석
- 예측 정확도 측정 및 개선
- 계절성 및 트렌드 분석
- S&OP(Sales & Operations Planning) 지원

## 🔧 주요 Excel 수식

### 1. 이동평균
```excel
=AVERAGE(OFFSET(B2,-기간수+1,0,기간수,1))
```

### 2. 지수평활법
```excel
// 단순지수평활
=α*실제수요+(1-α)*이전예측

// 이중지수평활 (Holt 방법)
=Level+Trend
Level = α*실제수요+(1-α)*(이전Level+이전Trend)
Trend = β*(Level-이전Level)+(1-β)*이전Trend
```

### 3. 계절지수 계산
```excel
=AVERAGE(IF(MONTH(날짜범위)=해당월,실제수요/트렌드,""))
```

### 4. MAPE 계산
```excel
=AVERAGE(ABS((실제수요-예측수요)/실제수요))*100
```

### 5. 추적신호
```excel
=SUM(오차범위)/AVERAGE(ABS(오차범위))
```

## 📈 활용 가이드

### 1. 데이터 준비
1. **충분한 과거 데이터**: 최소 24개월, 계절성 있는 경우 36개월
2. **데이터 품질 확인**: 이상치, 결측치 처리
3. **외부요인 정리**: 프로모션, 경쟁사 동향, 경제지표 등
4. **데이터 전처리**: 계절조정, 이상치 제거

### 2. 모델 선택
- **안정적 수요**: 이동평균, 단순지수평활
- **트렌드 있는 수요**: 이중지수평활, 선형회귀
- **계절성 있는 수요**: 삼중지수평활, 계절분해
- **복잡한 패턴**: 다중회귀, 시계열 분해

### 3. 예측 프로세스
1. **월초**: 전월 실적 업데이트
2. **월중**: 예측 모델 재계산
3. **월말**: 다음달 예측 확정
4. **분기말**: 모델 성능 평가 및 조정

## 💡 예측 정확도 개선 팁

### 데이터 품질 향상
1. **실시간 데이터**: 일일/주간 단위 모니터링
2. **외부 데이터**: 경제지표, 날씨, 경쟁사 정보
3. **정성적 정보**: 영업팀 인사이트, 고객 피드백
4. **데이터 검증**: 이상치 탐지 및 수정

### 모델 고도화
1. **앙상블 방법**: 여러 모델의 가중평균
2. **적응적 파라미터**: 계절별 다른 α, β, γ 값
3. **구간별 모델**: 수요 수준별 다른 모델 적용
4. **머신러닝**: Python R 연동 고급 분석

### 조직적 개선
1. **정기 리뷰**: 월간 예측 정확도 회의
2. **크로스 체크**: 다부서 검증 프로세스
3. **지속적 학습**: 오차 원인 분석 및 개선
4. **시스템 연동**: ERP, BI 도구 활용

## 📊 성과 측정

### 목표 지표
- **MAPE**: 10% 이하 (우수), 15% 이하 (양호)
- **편향도**: ±5% 이내
- **예측 안정성**: 월간 변동계수 20% 이하
- **적시성**: 매월 5일 이내 예측 완료

### 벤치마킹
- **업계 평균**: 동일 업종 MAPE 수준
- **경쟁사 대비**: 예측 정확도 및 대응 속도
- **글로벌 모범**: 선진 기업 사례 연구
