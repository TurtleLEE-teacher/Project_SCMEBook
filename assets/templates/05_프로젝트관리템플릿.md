# SCM 프로젝트 관리 템플릿 (SCM Project Management Template)

## 📋 개요
SCM 개선 프로젝트를 체계적으로 관리하기 위한 종합 관리 도구입니다. 프로젝트 계획부터 실행, 모니터링, 완료까지 전 단계를 효과적으로 관리할 수 있습니다.

## 🎯 사용 목적
- SCM 개선 프로젝트 통합 관리
- 일정, 예산, 리소스 관리
- 리스크 식별 및 대응
- 이해관계자 커뮤니케이션
- 성과 측정 및 평가

## 🔧 주요 Excel 수식

### 1. 진행률 계산
```excel
=SUMPRODUCT(개별진행률,가중치)/SUM(가중치)
```

### 2. 일정 상태 판정
```excel
=IF(오늘날짜>계획종료일,"지연",IF(진행률<예상진행률,"주의","정상"))
```

### 3. 리스크 위험도 계산
```excel
=INDEX(위험도매트릭스,MATCH(확률,확률범위),MATCH(영향도,영향도범위))
```

### 4. 예산 효율성
```excel
=집행예산/계획예산*100
```

### 5. EVM 지표
```excel
// CPI (Cost Performance Index)
=EV/AC

// SPI (Schedule Performance Index)  
=EV/PV
```

## 📈 활용 가이드

### 1. 프로젝트 초기화
1. **프로젝트 헌장 작성**: 목적, 범위, 제약사항
2. **WBS 작성**: 작업 분해 및 담당자 배정
3. **일정 계획**: 의존관계, 마일스톤 설정
4. **리소스 계획**: 인력, 예산 배정

### 2. 실행 및 통제
1. **주간 진행 회의**: 진행률 점검, 이슈 해결
2. **월간 스티어링**: 경영진 보고, 중요 의사결정
3. **리스크 리뷰**: 신규 리스크 식별, 대응 상태 점검
4. **품질 게이트**: 단계별 품질 기준 통과 확인

### 3. 종료 및 평가
1. **산출물 인수**: 최종 결과물 검수
2. **성과 평가**: KPI 달성도, ROI 측정
3. **교훈 학습**: 성공/실패 요인 분석
4. **지식 이관**: 문서화, 인수인계

## 💡 프로젝트 성공 팁

### 계획 단계
1. **명확한 목표**: SMART 기준 목표 설정
2. **충분한 분석**: 현황 진단, 요구사항 분석
3. **현실적 계획**: 여유분 반영, 리스크 고려
4. **이해관계자 정렬**: 기대치 관리, 합의 도출

### 실행 단계
1. **적극적 소통**: 투명한 정보 공유
2. **신속한 대응**: 이슈 발생 시 즉시 대응
3. **품질 우선**: 단계별 품질 확보
4. **변화 관리**: 사용자 저항 최소화

### 통제 단계
1. **정기적 모니터링**: 주간/월간 점검
2. **데이터 기반**: 객관적 지표 활용
3. **예방적 관리**: 문제 발생 전 대응
4. **유연한 조정**: 상황 변화 대응

## 📊 성공 지표

### 프로젝트 성공 기준
- **일정**: 계획 대비 ±10% 이내
- **예산**: 계획 대비 105% 이내
- **품질**: 목표 품질 지표 달성
- **만족도**: 이해관계자 만족도 4.0 이상

### 벤치마킹
- **성공률**: 업계 평균 대비 상위 25%
- **ROI**: 3년 내 투자 회수
- **효과 지속성**: 1년 후에도 80% 효과 유지
