# 비용 분석 템플릿 (Cost Analysis Template)

## 📋 개요
SCM 전반의 비용을 체계적으로 분석하고 최적화하기 위한 종합 분석 도구입니다. 총 소유 비용(TCO), 활동 기반 원가(ABC), 비용 동인 분석을 통해 효과적인 비용 관리를 수행할 수 있습니다.

## 🎯 사용 목적
- 총 소유 비용(TCO) 분석
- 활동별 비용 분석 및 최적화
- 공급업체별 비용 비교
- 물류 비용 최적화
- 비용 절감 기회 식별

## 📊 Excel 템플릿 구조

### Sheet 1: 비용 대시보드
```
전체 비용 현황:
A1: 총 조달 비용         B1: [자동계산] 억원
A2: 총 물류 비용         B2: [자동계산] 억원  
A3: 총 재고 비용         B3: [자동계산] 억원
A4: 총 품질 비용         B4: [자동계산] 억원
A5: 총 SCM 비용          B5: [자동계산] 억원

비용 비율:
A7: 매출 대비 SCM 비용   B7: [자동계산] %
A8: 직접비 vs 간접비     B8: [자동계산] 
A9: 고정비 vs 변동비     B9: [자동계산]

차트 영역:
- 비용 구조 파이 차트
- 월별 비용 추이
- 공급업체별 비용 분포
- 비용 절감 현황
```

## 🔧 주요 Excel 수식

### 1. 단위당 비용 계산
```excel
=총비용/총수량
```

### 2. 비중 계산
```excel
=개별비용/총비용*100
```

### 3. TCO 현재가치 계산
```excel
=PV(할인율, 연수, -연간비용, -초기비용)
```

### 4. 재고 보유 비용
```excel
=평균재고가치*보유비용율
```

### 5. ROI 계산
```excel
=(연간절감액-투자비용)/투자비용*100
```

## 📈 활용 가이드

### 1. 데이터 수집
1. **회계 시스템**: 기본 비용 데이터
2. **ERP 시스템**: 운영 활동 데이터  
3. **물류 시스템**: 운송, 창고 비용
4. **품질 시스템**: 품질 관련 비용

### 2. 분석 순서
1. **비용 분류**: 직접/간접, 고정/변동, 활동별
2. **비용 배분**: 적절한 동인 기준 설정
3. **벤치마킹**: 업계 평균, 모범 사례
4. **개선 기회**: 비용 절감 포인트 식별

### 3. 정기 리포팅
- **월간**: 주요 비용 지표 모니터링
- **분기**: 비용 구조 변화 분석
- **년간**: TCO 전면 재검토

## 💡 비용 최적화 전략

### 조달 비용 절감
1. **공급업체 통합**: 규모의 경제 실현
2. **장기 계약**: 볼륨 디스카운트 확보
3. **글로벌 소싱**: 저비용 지역 활용
4. **전자 조달**: 프로세스 효율화

### 물류 비용 절감
1. **네트워크 최적화**: 거점 통합, 경로 최적화
2. **공동 물류**: 동종업계 협력
3. **자동화**: 창고 자동화, 시스템 고도화
4. **모드 최적화**: 운송 수단 최적 조합

### 재고 비용 절감
1. **적정 재고**: EOQ, 안전재고 최적화
2. **VMI**: 공급업체 관리 재고
3. **JIT**: 필요 시점 공급
4. **ABC 관리**: 차별화된 재고 정책

## 📊 성과 측정 지표

### 핵심 KPI
- **총 SCM 비용/매출**: 15% 이하 목표
- **비용 절감률**: 전년 대비 5% 이상
- **단위당 물류비**: 지속적 감소 추세
- **재고 회전율**: 12회/년 이상

### 벤치마킹 지표
- **업계 평균 대비**: 상위 25% 목표
- **글로벌 모범**: 선진 기업 90% 수준
- **비용 구조**: 직접비 80%, 간접비 20%
