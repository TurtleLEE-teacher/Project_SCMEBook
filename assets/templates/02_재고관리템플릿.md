# 재고 관리 템플릿 (Inventory Management Template)

## 📋 개요
효율적인 재고 관리를 위한 종합 관리 도구입니다. 적정재고 계산, ABC 분석, 재고 회전율 분석 등을 통해 최적의 재고 수준을 유지할 수 있습니다.

## 🎯 사용 목적
- 적정 재고 수준 계산
- 재고 회전율 분석
- ABC 분석을 통한 중점 관리 품목 식별
- 재고 비용 최적화
- 재고 부족 및 과다 재고 방지

## 🔧 주요 Excel 수식

### 1. EOQ 계산
```excel
=SQRT(2*연간수요*주문비용/보관비용)
```

### 2. 안전재고 계산
```excel
=Z값*SQRT(리드타임)*수요표준편차
```

### 3. 재고 회전율
```excel
=연간출고량/평균재고
```

### 4. ABC 분류 자동화
```excel
=IF(누적비율<=80,"A",IF(누적비율<=95,"B","C"))
```

### 5. 재주문 알림
```excel
=IF(현재고<=재주문점,"주문필요","")
```

## 📈 활용 가이드

### 1. 초기 설정
1. **품목 마스터 등록**: 관리 대상 품목 정보 입력
2. **수요 데이터 입력**: 최근 12개월 실적 데이터
3. **비용 파라미터 설정**: 주문비용, 보관비용 등
4. **서비스 레벨 설정**: 목표 서비스 수준 (95-99%)

### 2. 정기 운영
1. **주간**: 현재고 업데이트, 주문 제안 검토
2. **월간**: 재고 회전율 분석, ABC 분석 업데이트
3. **분기**: 적정재고 수준 재계산
4. **연간**: 전체 재고 정책 재검토

### 3. 의사결정 지원
- **주문 시점**: 재주문점 도달 시 자동 알림
- **주문 수량**: EOQ 기반 최적 주문량 제안
- **우선순위**: ABC 등급별 차별화된 관리
- **비용 최적화**: 재고 비용 vs 서비스 레벨 균형

## 💡 실무 적용 팁

### 효과적인 재고 관리
1. **데이터 정확성**: 정확한 재고 데이터가 모든 분석의 기본
2. **정기적 검토**: 수요 패턴 변화에 따른 파라미터 조정
3. **공급업체 협력**: 리드타임 단축 및 유연성 확보
4. **시스템 활용**: ERP 연동을 통한 실시간 관리

### 일반적인 함정 방지
- ❌ 과거 데이터만을 기반한 계획
- ❌ 모든 품목에 동일한 관리 수준 적용
- ❌ 비용 고려 없는 서비스 레벨 설정
- ❌ 공급업체 리드타임 변동 미반영
