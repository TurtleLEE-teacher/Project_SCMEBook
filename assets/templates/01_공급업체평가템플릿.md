# 공급업체 평가 템플릿 (Supplier Evaluation Template)

## 📋 개요
공급업체 선정 및 정기 평가를 위한 종합 평가 도구입니다. 품질, 납기, 가격, 서비스 등 핵심 평가 요소를 체계적으로 측정할 수 있습니다.

## 🎯 사용 목적
- 신규 공급업체 선정
- 기존 공급업체 정기 평가
- 공급업체 등급 관리
- 개선 영역 식별

## 📊 Excel 템플릿 구조

### Sheet 1: 평가 대시보드
```
A1: 공급업체명
B1: 평가일자
C1: 평가자
D1: 총점
E1: 등급

A3: 종합 점수: [자동계산]
B3: 등급: [A/B/C/D 자동 분류]
C3: 추천도: [추천/보류/비추천]
```

### Sheet 3: 등급 기준표

| 등급 | 점수 범위 | 분류 | 조치사항 |

## 🔧 Excel 수식 예시

### 자동 계산 수식
```excel
// 총점 계산
=SUMPRODUCT(점수범위, 가중치범위)

// 등급 자동 분류
=IF(총점>=90,"A",IF(총점>=80,"B",IF(총점>=70,"C",IF(총점>=60,"D","F"))))

// 추천도 계산
=IF(총점>=80,"추천",IF(총점>=70,"보류","비추천"))

// 개선 필요 영역 식별
=IF(항목점수<3,"개선필요","양호")
```

## 📈 활용 가이드

### 1. 평가 진행 순서
1. **사전 준비**: 평가 기준 및 가중치 확인
2. **자료 수집**: 공급업체 성과 데이터 수집
3. **평가 실시**: 각 항목별 점수 입력
4. **결과 분석**: 총점, 등급, 개선 영역 확인
5. **피드백**: 공급업체와 결과 공유 및 개선 계획 수립

### 2. 주의사항
- 객관적 데이터를 기반으로 평가
- 여러 평가자의 의견 수렴
- 정기적(연 2회) 평가 실시
- 평가 결과의 일관성 유지

### 3. 커스터마이징
- 업종별 특성에 따른 가중치 조정
- 평가 항목 추가/삭제
- 등급 기준 조정
- 자동화 수식 확장

## 💡 실무 팁

### 효과적인 평가를 위한 팁
1. **데이터 기반 평가**: 주관적 판단보다는 객관적 데이터 활용
2. **벤치마킹**: 다른 공급업체와의 비교 분석
3. **트렌드 분석**: 과거 평가 결과와의 비교를 통한 개선도 측정
4. **협업적 접근**: 공급업체와 함께 개선 방안 모색

### 일반적인 실수 방지
- ❌ 한 번의 평가로 모든 것을 판단
- ❌ 가격만을 중시한 평가
- ❌ 정성적 평가의 일관성 부족
- ❌ 피드백 없는 일방적 평가
