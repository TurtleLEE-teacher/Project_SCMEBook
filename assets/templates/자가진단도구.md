# SCM 전문가 자가 진단 도구

## 📋 개요
SCM 분야에서 자신의 현재 역량 수준을 객관적으로 평가하고, 성장 방향을 설정할 수 있는 종합 진단 도구입니다. 주니어부터 컨설턴트 레벨까지 단계별 역량을 측정합니다.

## 🎯 사용 방법
1. 각 영역별로 자신에게 해당하는 점수를 체크하세요 (1~5점)
2. 영역별 평균 점수를 계산하세요
3. 종합 점수를 바탕으로 현재 레벨을 확인하세요
4. 개선이 필요한 영역을 파악하고 학습 계획을 수립하세요

## 📊 평가 척도
- **5점**: 전문가 수준 (업계 상위 10%, 타인 지도 가능)
- **4점**: 숙련 수준 (독립적 업무 수행, 문제 해결 가능)
- **3점**: 보통 수준 (기본 업무 수행, 약간의 지원 필요)
- **2점**: 초급 수준 (교육/지원 하에 업무 수행)
- **1점**: 입문 수준 (기초 지식 부족, 체계적 학습 필요)
