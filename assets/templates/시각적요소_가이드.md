# SCM 프로세스 다이어그램 및 인포그래픽 가이드

## 📋 개요
SCM 전자책에서 활용할 수 있는 프로세스 다이어그램과 인포그래픽의 설계 가이드입니다. 복잡한 SCM 개념을 시각적으로 이해하기 쉽게 표현하는 방법을 제시합니다.

## 🔄 1. SCM 핵심 프로세스 다이어그램

### A. 전체 공급망 프로세스 플로우
```
[공급업체] → [제조사] → [유통업체] → [소매업체] → [최종고객]
     ↑         ↑          ↑           ↑           ↑
   원자재     생산계획    재고관리     주문처리    고객서비스
   조달      품질관리    물류관리     배송관리    A/S
   계약관리   생산실행    창고관리     결제처리    마케팅

📊 정보 흐름 (양방향)
💰 자금 흐름 (역방향)
📦 물류 흐름 (정방향)
```

### B. 구매 프로세스 상세 플로우
```
1단계: 구매요청        2단계: 공급업체선정      3단계: 주문처리
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│ • 수요 발생     │ ──→ │ • 업체 조사     │ ──→ │ • 주문서 작성   │
│ • 구매요청서    │     │ • 견적 요청     │     │ • 계약 체결     │
│ • 예산 승인     │     │ • 협상 진행     │     │ • 주문 발송     │
└─────────────┘     └─────────────┘     └─────────────┘
        │                      │                      │
        ▼                      ▼                      ▼
4단계: 납품관리        5단계: 검수처리         6단계: 결제처리
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│ • 납기 추적     │     │ • 입고 검수     │     │ • 검수 완료     │
│ • 진도 관리     │ ←── │ • 품질 검사     │ ←── │ • 대금 지급     │
│ • 변경 관리     │     │ • 불량 처리     │     │ • 정산 완료     │
└─────────────┘     └─────────────┘     └─────────────┘
```

```
        📊 수요 예측
           ↓
    📋 재고 계획 수립
           ↓
    🛒 주문 의사결정 ←─────┐
           ↓               │
    📦 재고 입고           │
           ↓               │
    📊 재고 모니터링        │

```

## 📈 4. 프로세스 개선 Before/After 비교

### A. 구매 프로세스 개선 사례
```
📉 BEFORE (개선 전)
수동 프로세스: 평균 7-10일 소요
┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐
│요청서│→│승인 │→│견적 │→│검토 │→│주문 │
│작성 │ │대기 │ │수집 │ │회의 │ │처리 │
│1일 │ │2-3일│ │2-3일│ │1-2일│ │1일 │
└─────┘ └─────┘ └─────┘ └─────┘ └─────┘

❌ 문제점:
• 수작업으로 인한 지연
• 승인 대기 시간 과다
• 정보 추적 어려움
• 오류 발생 가능성

📈 AFTER (개선 후)
자동화 프로세스: 평균 3-4일 소요
┌─────┐ ┌─────┐ ┌─────┐
│온라인│→│자동 │→│전자 │
│요청 │ │승인 │ │주문 │
│0.5일│ │1일  │ │1일  │
└─────┘ └─────┘ └─────┘

✅ 개선 효과:
• 처리 시간 60% 단축
• 오류율 80% 감소
• 비용 절감 15%
• 만족도 90% 향상
```

1. **간결성**: 핵심 정보만 선별하여 표현
2. **일관성**: 색상, 폰트, 스타일 통일
3. **명확성**: 직관적 이해가 가능한 구조
4. **매력성**: 시각적으로 흥미로운 디자인

### 📊 독자 친화적 설계
1. **단계별 구성**: 복잡한 내용을 단계별로 분해
2. **시각적 계층**: 크기와 색상으로 중요도 표현
3. **상호 참조**: 텍스트와 시각 요소의 연결
4. **실용성**: 실무에서 활용 가능한 정보 제공
