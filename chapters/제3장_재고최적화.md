## 📖 도입부: 개인 경험담 (50%)

### 350억 원의 재고, 그리고 깨달음

외자구매팀에서 3개월째 일하던 어느 날, 사장님이 긴급 회의를 소집했다.

"우리 회사 재고가 350억 원이다. 작년 동기 대비 40% 증가했다. 이게 정상인가?"

회의실에 묵직한 침묵이 흘렀다. 그때까지 나는 재고를 단순히 '필요할 때 쓰는 물건들'이라고 생각했다. 하지만 그 순간 깨달았다. **재고는 돈이다. 창고에 잠자고 있는 현금이다.**

### 재고의 역설: 많으면 문제, 적어도 문제

"재고가 많으면 뭐가 문제인가요?" 신입사원이었던 내가 조심스럽게 물었다.

CFO가 한숨을 쉬며 답했다:
- "자본 비용만 연간 17억 원이다 (5% 금리 기준)"
- "창고 임대료, 보험료까지 합치면 35억 원"
- "게다가 폐기되는 제품도 매년 10억 원 어치"

그런데 영업팀장이 반박했다:
- "재고가 부족하면 결품으로 고객을 잃는다"
- "긴급 주문 시 높은 비용을 지불해야 한다"
- "시장 기회를 놓칠 수도 있다"

그날 나는 깨달았다. **재고 관리는 줄타기와 같다. 적정선을 찾는 것이 핵심이다.**

### 첫 번째 재고 분석 프로젝트

팀장이 나에게 특별 미션을 주었다.
"이선생, 우리 재고를 한번 분석해봐. 어떤 품목이 문제인지 찾아보자."

엑셀과 씨름하며 3주간 분석한 결과는 충격적이었다:
- **상위 20% 품목**이 전체 재고 가치의 **80%**를 차지
- **하위 60% 품목**은 재고 가치의 **5%**에 불과
- 일부 품목은 **2년 넘게** 창고에 잠자고 있었다

"이게 바로 ABC 분석이구나!" 그때부터 재고 관리의 과학적 접근법에 빠져들기 시작했다.

## 🎯 이론부: 핵심 개념과 프레임워크 (50%)

### 재고 관리의 기본 개념

내가 첫 재고 분석을 통해 배운 것은 **재고 관리가 단순한 물건 세기가 아니라 전략적 의사결정**이라는 점이었다.

#### 1. 재고의 4가지 유형

| 재고 유형 | 정의 | 목적 | 관리 포인트 |
|----------|------|------|-------------|
| **순환재고** | 정기적 주문으로 발생 | 운영 효율성 | EOQ 최적화 |
| **안전재고** | 불확실성 대비 버퍼 | 서비스 레벨 보장 | 적정 수준 설정 |
| **계절재고** | 계절적 수요 변동 대비 | 수요 집중 대응 | 예측 정확도 향상 |
| **투기재고** | 가격 상승 대비 | 비용 절감 | 시장 분석 |

#### 2. 재고 비용의 구성

**재고 보유 비용 = 자본비용 + 창고비용 + 서비스비용 + 리스크비용**

```
실제 계산 예시 (연간 기준):
┌─────────────┬─────────┬──────────┐
│ 비용 구분   │ 비율(%) │ 금액(억원) │
├─────────────┼─────────┼──────────┤
│ 자본 비용   │   5.0   │    17.5   │
│ 창고 비용   │   2.0   │     7.0   │
│ 보험 비용   │   0.5   │     1.8   │
│ 관리 비용   │   1.0   │     3.5   │
│ 손실 비용   │   1.5   │     5.3   │
├─────────────┼─────────┼──────────┤
│ 총 보유비용 │  10.0   │    35.0   │
└─────────────┴─────────┴──────────┘
```

#### 3. ABC 분석법

내가 처음 발견한 80-20 법칙이 바로 ABC 분석이었다.

**분류 기준**:
- **A등급**: 상위 20% 품목 (재고 가치의 80%)
- **B등급**: 중간 30% 품목 (재고 가치의 15%)  
- **C등급**: 하위 50% 품목 (재고 가치의 5%)

**등급별 관리 전략**:
```
A등급 (Critical Items):
├ 일일 모니터링
├ 정밀한 수요 예측
├ 안전재고 최소화
└ 공급업체 다변화

B등급 (Important Items):
├ 주간 모니터링
├ 표준 예측 모델
├ 적정 안전재고
└ 정기 성과 검토

C등급 (Standard Items):
├ 월간 모니터링
├ 간단한 예측 기법
├ 높은 안전재고 허용
└ 단순 관리 방식
```

#### 4. EOQ (Economic Order Quantity) 모델

**기본 공식**:
```
EOQ = √(2 × D × S / H)

여기서:
D = 연간 수요량
S = 주문 비용 (1회당)
H = 연간 보유 비용 (단위당)
```

**실제 계산 예시**:
```
품목: A-100 (주요 부품)
D = 12,000개/년
S = 500,000원/주문
H = 20,000원/개/년

EOQ = √(2 × 12,000 × 500,000 / 20,000)
    = √(600,000,000)
    = 774개

→ 한 번에 774개씩 주문하는 것이 최적
```

#### 5. 안전재고 계산

**기본 공식**:
```
안전재고 = Z × σLT

여기서:
Z = 서비스 레벨에 해당하는 표준정규분포 값
σLT = 리드타임 동안의 수요 표준편차
```

**서비스 레벨별 Z값**:
- 90%: Z = 1.28
- 95%: Z = 1.65  
- 98%: Z = 2.05
- 99%: Z = 2.33

### 6. 고급 재고 관리 기법

#### A. VMI (Vendor Managed Inventory)
**공급업체 관리 재고 시스템**

장점:
- 재고 비용 절감 (평균 20-30%)
- 결품률 감소 (95% → 99% 서비스 레벨)
- 관리 업무 부담 경감

단점:
- 공급업체 의존성 증가
- 정보 공유 필요
- 통제권 일부 상실

#### B. JIT (Just-In-Time)
**적시 생산 시스템**

핵심 원칙:
- 필요한 것을
- 필요한 때에
- 필요한 만큼만

성공 조건:
- 안정적 공급업체
- 짧은 리드타임
- 높은 품질 수준
- 유연한 생산 시스템

#### C. 디지털 재고 관리

**IoT 센서 활용**:
- 실시간 재고 수준 모니터링
- 자동 발주 시스템
- 예측 정비 (Predictive Maintenance)

**AI/ML 예측**:
- 수요 패턴 학습
- 계절성/트렌드 자동 인식
- 동적 안전재고 조정

### 7. 글로벌 재고 관리의 복잡성

#### A. 다국가 재고 관리 전략

**다국가 운영의 도전과제**:
외자구매 경험을 통해 깨달은 것은 **글로벌 공급망에서 재고 관리의 복잡성**이었다. 단일 국가 내 재고 관리와는 완전히 다른 차원의 고려사항들이 있었다.

**주요 고려사항들**:
```
지역별 차이점:
- 수요 패턴의 지역적 특성
- 문화적 차이에 따른 제품 선호도
- 경제적 상황과 구매력 차이
- 법규 및 규제 환경의 상이함

운영적 복잡성:
- 다양한 통화와 환율 변동 리스크
- 국가별 세무 및 관세 정책
- 지역별 공급업체 네트워크
- 다국가 간 물류 및 운송 체계
```

**글로벌 재고 최적화 원칙**:
1. **지역화 vs 표준화**: 글로벌 효율성과 지역 적응성의 균형
2. **리스크 분산**: 지역별 리스크를 고려한 재고 배치
3. **스케일 이코노미**: 규모의 경제를 활용한 비용 절감
4. **유연성 확보**: 변화하는 시장 상황에 대한 빠른 대응

#### B. 디지털 혁신과 재고 관리

**AI와 머신러닝의 활용**:
최근 몇 년간 재고 관리 분야에서 **인공지능과 머신러닝 기술**의 도입이 급속히 진행되고 있다. 과거의 정적인 재고 관리에서 동적이고 예측적인 관리로의 전환이 핵심이다.

**주요 적용 분야**:
- **수요 예측 정확도 향상**: 복잡한 패턴 인식과 예측
- **동적 안전재고 조정**: 실시간 리스크 평가 기반 최적화
- **자동 발주 시스템**: AI 기반 주문 최적화
- **이상 감지**: 비정상적 재고 움직임 조기 탐지

**실시간 가시성과 투명성**:
```
전통적 재고 관리: 주기적 점검 → 사후 대응
↓
현대적 재고 관리: 실시간 모니터링 → 예방적 대응
```

### 8. 지속가능한 재고 관리

#### A. 환경적 고려사항

**ESG와 재고 관리**:
최근 기업들의 **ESG(환경, 사회, 지배구조)** 경영이 중요해지면서, 재고 관리에서도 환경적 영향을 고려해야 하는 시대가 되었다.

**주요 고려사항들**:
- **탄소 발자국 최소화**: 운송과 보관 과정의 환경 영향
- **폐기물 감소**: 유통기한 관리와 순환 경제 원칙
- **지속가능한 포장**: 친환경 포장재 사용
- **공급업체 ESG 평가**: 지속가능한 공급망 구축

#### B. 사회적 책임과 재고 관리

**윤리적 재고 관리**:
- **공정 거래**: 공급업체와의 상생 관계
- **지역 사회 기여**: 로컬 소싱과 지역 경제 활성화
- **인권 존중**: 공급망 전반의 인권 보호
- **투명성 확보**: 이해관계자에 대한 정보 공개

### 9. 재고 관리 전문가로의 성장 경로

#### A. 역량 개발 로드맵

**초급 단계 (1-2년)**:
```
기초 역량:
- ABC 분석과 EOQ 모델 이해
- 기본적인 재고 시스템 운영
- 엑셀 기반 데이터 분석
- 재고 관련 KPI 이해
```

**중급 단계 (3-5년)**:
```
응용 역량:
- 고급 예측 모델 활용
- 다중 사이트 재고 최적화
- 공급업체와의 협업적 재고 관리
- 재고 관리 시스템 설계 참여
```

**고급 단계 (5년 이상)**:
```
전략적 역량:
- 전사 재고 전략 수립
- 디지털 혁신 리드
- 글로벌 재고 네트워크 설계
- 조직 변화 관리
```

#### B. 지속적 학습과 발전

**학습 리소스 활용**:
- **전문 자격증**: APICS CSCP, CLTD 등
- **온라인 코스**: Coursera, edX 플랫폼
- **컨퍼런스 참석**: 업계 동향과 네트워킹
- **벤치마킹**: 선진 기업 사례 연구

**실무 경험 확장**:
- **프로젝트 리드**: 재고 최적화 프로젝트 주도
- **크로스 펑셔널 협업**: 다른 부서와의 협력 경험
- **글로벌 경험**: 해외 법인과의 협업
- **멘토링**: 후배 직원 가이드 경험
