## 📖 도입부: 개인 경험담 (50%)

### 2억 원의 재고, 그리고 잘못된 예측

외자구매팀에서 일한 지 1년이 되던 어느 날, 사장님이 긴급 회의를 소집했다.

"이선생, 우리가 작년에 대량 주문한 A부품 있지? 지금 창고에 얼마나 남아있나?"

"네... 약 2억 원어치가 남아있습니다."

회의실이 조용해졌다. 당초 예측했던 수요의 절반도 되지 않았던 것이다.

### 예측의 함정에 빠지다

그때 우리가 범한 실수는 단순했지만 치명적이었다:

1. **과거 데이터만 믿었다**: "작년에 잘 팔렸으니 올해도..."
2. **시장 변화를 무시했다**: 경쟁업체의 신제품 출시
3. **계절성을 고려하지 않았다**: 특정 시기에만 수요가 몰린다는 사실
4. **다양한 의견을 수렴하지 않았다**: 영업팀의 경고를 무시

"예측이 틀리면 재고가 되고, 재고는 곧 손실이구나." 그날 뼈저리게 깨달았다.

### 수요 예측의 첫 번째 교훈

팀장이 나에게 특별 미션을 주었다. "이선생, 앞으로 우리 수요 예측 담당해봐. 이런 일이 다시 생기지 않도록."

처음에는 막막했다. 하지만 3개월간 여러 방법을 시도해보면서 깨달았다:

**"완벽한 예측은 없다. 하지만 체계적인 접근으로 정확도를 높일 수는 있다."**

첫 번째 성공 사례는 B부품이었다. 다양한 데이터를 종합 분석한 결과, 예측 정확도를 30%에서 85%로 향상시켰다. 그때부터 수요 예측의 과학에 빠져들기 시작했다.

## 🎯 이론부: 핵심 개념과 프레임워크 (50%)

### 수요 예측의 기본 개념

나의 첫 번째 실패와 성공 경험을 통해 배운 것은 **수요 예측이 단순한 추측이 아니라 과학적 방법론**이라는 점이었다.

#### 1. 수요 예측의 유형

| 예측 유형 | 기간 | 목적 | 방법론 |
|----------|------|------|--------|
| **단기 예측** | 1일~3개월 | 생산 계획, 재고 관리 | 시계열 분석, 지수평활법 |
| **중기 예측** | 3개월~2년 | 예산 계획, 자원 배분 | 회귀분석, 계절성 분석 |
| **장기 예측** | 2년 이상 | 전략 수립, 투자 결정 | 시나리오 분석, 델파이 기법 |

#### 2. 수요의 구성 요소

**수요 = 기본 수요 + 트렌드 + 계절성 + 불규칙 변동**

```
실제 수요 분해 예시:
┌─────────────────┐
│ 월별 판매량    │ = 1,000 + (10 × 월수) + 계절지수 + 오차
│ (기본 1,000개)  │   ↑        ↑           ↑        ↑
│                 │ 기본수요   트렌드     계절성   불규칙
└─────────────────┘
```

#### 3. 예측 정확도 측정

**핵심 지표들**:

**A. MAPE (Mean Absolute Percentage Error)**
```
MAPE = (1/n) × Σ|실제값 - 예측값|/실제값 × 100

예시:
실제: 1,000개, 예측: 850개
APE = |1,000 - 850|/1,000 × 100 = 15%
```

**B. MAD (Mean Absolute Deviation)**
```
MAD = (1/n) × Σ|실제값 - 예측값|

장점: 단위가 원래 데이터와 동일
활용: 안전재고 계산에 유용
```

**C. 예측 정확도 평가 기준**
- **90% 이상**: 우수
- **80-89%**: 양호  
- **70-79%**: 보통
- **70% 미만**: 개선 필요

### 4. 정성적 예측 방법

#### A. 델파이 기법 (Delphi Method)
**정의**: 전문가들의 의견을 반복적으로 수렴하여 합의에 도달하는 방법

**실행 과정**:
```
1단계: 전문가 패널 구성 (5-10명)
↓
2단계: 첫 번째 설문 실시
↓
3단계: 결과 정리 및 피드백
↓
4단계: 두 번째 설문 실시 (조정된 의견)
↓
5단계: 합의 도달 또는 반복
```

#### B. 시장 조사 및 고객 설문
**핵심 질문 유형**:
- **구매 의도**: "향후 6개월 내 구매 계획이 있습니까?"
- **수량 예상**: "예상 구매 수량은 얼마입니까?"
- **타이밍**: "언제 구매하실 예정입니까?"
- **영향 요인**: "구매 결정에 영향을 미치는 요인은?"

#### C. 영업팀 종합 의견 (Sales Force Composite)
**장점**:
- 고객과 직접 접촉하는 현장 경험
- 시장 변화에 대한 빠른 감지
- 지역별/고객별 세부 정보 보유

**단점**:
- 개인적 편견 개입 가능
- 과도하게 낙관적/비관적 성향
- 정량적 분석 부족

### 5. 정량적 예측 방법

#### A. 이동평균법 (Moving Average)
**기본 공식**:
```
3개월 이동평균 = (1월 + 2월 + 3월) / 3

예시:
1월: 1,000개
2월: 1,200개  
3월: 800개
→ 4월 예측: (1,000 + 1,200 + 800) / 3 = 1,000개
```

**가중 이동평균**:
```
최근 데이터에 더 높은 가중치 부여
가중이동평균 = (w1×최근값 + w2×이전값 + w3×과거값) / (w1+w2+w3)

예시: w1=0.5, w2=0.3, w3=0.2
→ 4월 예측 = (0.5×800 + 0.3×1,200 + 0.2×1,000) = 960개
```

#### B. 지수평활법 (Exponential Smoothing)
**단순 지수평활법**:
```
F(t+1) = α × A(t) + (1-α) × F(t)

여기서:
F(t+1) = 다음 기간 예측값
α = 평활상수 (0 < α < 1)
A(t) = 현재 실제값
F(t) = 현재 예측값
```

**이중 지수평활법 (홀트 방법)**:
```
트렌드가 있는 데이터 적합
L(t) = α × A(t) + (1-α) × [L(t-1) + T(t-1)]
T(t) = β × [L(t) - L(t-1)] + (1-β) × T(t-1)
F(t+m) = L(t) + m × T(t)

L(t): 레벨 성분
T(t): 트렌드 성분
β: 트렌드 평활상수
```

#### C. 회귀분석 (Regression Analysis)
**단순 회귀분석**:
```
Y = a + bX + e

여기서:
Y = 수요량 (종속변수)
X = 독립변수 (시간, 가격, 소득 등)
a = 절편
b = 기울기
e = 오차항
```

**다중 회귀분석 예시**:
```
수요량 = β0 + β1×가격 + β2×소득 + β3×광고비 + β4×경쟁사가격 + e

실제 적용 예:
월 수요량 = 1,500 - 2.5×가격 + 0.8×광고비 + 1.2×경쟁사가격
```

#### D. 시계열 분석
**ARIMA 모델 (AutoRegressive Integrated Moving Average)**:
```
ARIMA(p,d,q) 모델:
- p: 자기회귀(AR) 차수
- d: 차분(I) 차수  
- q: 이동평균(MA) 차수

적용 절차:
1단계: 데이터 정상성 확인
2단계: 차분을 통한 정상화
3단계: ACF/PACF 분석으로 p,q 결정
4단계: 모델 추정 및 검정
5단계: 예측 실시
```

### 6. 계절성 분석

#### A. 계절지수 계산
```
계절성 분해 과정:
1단계: 이동평균 계산 (트렌드 제거)
2단계: 계절-불규칙 성분 분리
3단계: 계절지수 계산
4단계: 정규화 (연평균 = 1.0)
```

**실제 계산 예시**:
| 월 | 실제값 | 이동평균 | 계절-불규칙 | 계절지수 |
|----|--------|----------|-------------|----------|
| 1월 | 800 | 1,000 | 0.80 | 0.82 |
| 2월 | 900 | 1,010 | 0.89 | 0.91 |
| 3월 | 1,200 | 1,020 | 1.18 | 1.15 |
| 12월 | 1,400 | 1,100 | 1.27 | 1.25 |

#### B. 계절성 적용 예측
```
조정된 예측값 = 기본 예측값 × 계절지수

예시:
3월 기본 예측: 1,000개
3월 계절지수: 1.15
→ 3월 최종 예측: 1,000 × 1.15 = 1,150개
```

### 7. 실무 경험을 통한 수요 예측 전문가로의 성장

#### A. 수요 예측 정확도 향상 여정

**첫 2년의 시행착오**:
수요 예측 업무를 시작하면서 가장 큰 어려움은 **이론과 현실의 괴리**였다. 대학에서 배운 통계 모델들은 완벽해 보였지만, 실제 비즈니스 환경에서는 예상치 못한 변수들이 끊임없이 발생했다.

```
초기 실패 패턴들:
- 과거 데이터만 맹신: "작년 패턴이 올해도 반복될 것"
- 이상값 무시: "이번만 특별한 상황이야"
- 정성적 정보 경시: "숫자가 모든 것을 말해준다"
- 단일 모델 의존: "이 모델이 최고야"
```

**3년차의 전환점**:
여러 번의 예측 실패를 겪으며 깨달은 것은 **완벽한 예측은 불가능하지만, 체계적인 접근으로 지속적인 개선은 가능하다**는 점이었다.

- **다중 모델 앙상블**: 여러 모델의 결과를 종합
- **정성적 조정**: 전문가 의견과 시장 정보 반영
- **실시간 모니터링**: 예측 결과와 실제값의 지속적 비교
- **빠른 수정**: 편차 발생 시 즉시 모델 재조정

#### B. 조직 내 수요 예측 시스템 구축

**부서간 협업 체계 구축**:
수요 예측의 정확도를 높이기 위해서는 **여러 부서의 정보와 전문성을 통합**하는 것이 핵심이었다.

**효과적인 협업 프로세스**:
```
월간 수요 예측 회의:
1주차: 시장 정보 수집 (영업팀, 마케팅팀)
2주차: 데이터 분석 및 모델링 (분석팀)
3주차: 결과 검토 및 조정 (전 부서)
4주차: 최종 확정 및 공유 (경영진)
```

**성공 요인들**:
- **데이터 표준화**: 모든 부서가 동일한 기준으로 데이터 수집
- **책임 명확화**: 부서별 역할과 책임 명확히 정의
- **성과 공유**: 예측 정확도 개선 효과를 조직 전체가 체감
- **지속적 학습**: 실패 사례 분석을 통한 조직 학습

#### C. 기술적 전문성의 진화

**Excel에서 AI까지의 여정**:
초기에는 **Excel 기반의 단순한 통계 분석**에서 시작했지만, 점차 **고급 분석 도구와 AI 기술**을 활용하게 되었다.

**기술 스택의 진화**:
```
1단계 (1-2년): Excel + 기본 통계
↓ 단순 이동평균, 회귀분석
2단계 (3-4년): R/Python + 고급 통계
↓ ARIMA, 지수평활법, 계절분해
3단계 (5년+): ML/AI + 실시간 시스템
↓ Random Forest, LSTM, 자동화 시스템
```

**각 단계별 핵심 학습 포인트**:
- **1단계**: 데이터의 기본적 패턴 이해와 시각화
- **2단계**: 통계적 모델링과 가설 검정
- **3단계**: 복잡한 패턴 인식과 자동화 시스템

### 8. 산업별 수요 예측 특성과 전략

#### A. 제조업에서의 수요 예측

**B2B 수요의 특수성**:
외자구매 경험을 통해 알게 된 것은 **제조업에서의 수요 예측은 최종 소비자 수요와 완전히 다른 특성**을 갖는다는 점이었다.

**핵심 차이점들**:
- **파생 수요**: 최종 제품 수요에서 역산되는 부품 수요
- **계획적 주문**: 고객사의 생산 계획에 따른 예측 가능한 패턴
- **벌크 오더**: 대량 주문으로 인한 수요의 집중성
- **장기 계약**: 연간 계약에 따른 상대적 안정성

**성공적인 B2B 수요 예측 전략**:
```
고객사 생산계획 분석:
├ 주요 고객사의 생산 스케줄 파악
├ 신제품 출시 일정과 런칭 수량 예측
├ 계절적 생산 패턴 분석
└ 경쟁사 동향과 시장 점유율 변화
```

#### B. 글로벌 시장에서의 수요 예측

**다국가 수요 패턴의 복잡성**:
해외 공급업체들과 일하면서 경험한 것은 **각 국가별로 완전히 다른 수요 패턴**을 보인다는 점이었다.

**지역별 고려사항들**:
- **문화적 차이**: 제품 선호도와 사용 패턴의 차이
- **경제적 사이클**: 각국의 경제 상황과 성장률
- **정치적 안정성**: 정치적 이벤트가 수요에 미치는 영향
- **기후와 계절성**: 지리적 위치에 따른 수요 시기 차이

#### C. 디지털 혁신과 수요 예측의 미래

**실시간 데이터의 활용**:
최근 몇 년간 **IoT 센서와 실시간 데이터 분석**이 수요 예측의 패러다임을 바꾸고 있다. 과거의 정적인 예측에서 동적이고 적응적인 예측으로의 전환이 핵심이다.

**혁신적 데이터 소스들**:
- **소셜미디어 감성 분석**: 제품에 대한 실시간 감정 변화
- **위치 기반 데이터**: 유동인구와 지역별 수요 패턴
- **날씨 정보**: 기상 변화가 수요에 미치는 즉각적 영향
- **경제 지표**: 거시경제 변화의 실시간 반영

**AI/ML의 실제 적용 경험**:
최근 프로젝트에서 **머신러닝 모델을 도입**한 결과, 예측 정확도가 평균 15% 향상되었다. 특히 **복잡한 비선형 패턴 인식**에서 기존 통계 모델보다 월등한 성능을 보였다.

### 9. 수요 예측 전문가로서의 역량 개발 로드맵

#### A. 기술적 역량 발전 경로

**단계별 역량 개발**:
