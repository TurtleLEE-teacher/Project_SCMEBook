## 📖 프로젝트 소개

**"SCM 현장에서 컨설팅까지: 이선생의 공급망 항해일지"**는 Supply Chain Management(SCM) 분야에서 주니어 담당자부터 컨설턴트까지의 성장 여정을 다룬 실무 중심의 전자책입니다.

### 🎯 대상 독자
- **초급자** (0-2년차): SCM 기초 이론과 실무 적응
- **중급자** (3-7년차): 고급 분석 기법과 리더십 개발  
- **고급자** (7년+ 리더/컨설턴트): 전략적 사고와 컨설팅 역량

### ✨ 특징
- 📱 **8가지 테마 접근**: 스토리, 캐릭터, 배경, 주제, 갈등, 감정, 상징, 구조
- 🎭 **스토리텔링 방식**: 실제 현장 경험담 기반
- 🛠️ **실무 중심**: 즉시 적용 가능한 도구와 템플릿
- 📊 **체계적 관리**: 노션 데이터베이스 활용

## 🎨 핵심 컨셉

### 8가지 테마 접근법
1. **📖 스토리**: 생생한 현장 경험담
2. **👥 캐릭터**: 다양한 이해관계자 관점  
3. **🌍 배경**: SCM 업계 환경과 맥락
4. **🎯 주제**: 핵심 학습 내용과 이론
5. **⚡ 갈등**: 실무 도전과제와 해결방안
6. **💭 감정**: 성장 과정의 심리적 변화
7. **🔮 상징**: SCM 개념의 시각적 표현
8. **🏗️ 구조**: 체계적인 학습 프레임워크

### 독자별 맞춤 학습 경로

#### 🌱 초급자 (0-2년차)
```
프롤로그 → 1-3장 → 용어집 → 4-6장
```
**포커스**: 기초 개념, 실무 적응, 마인드셋

#### 🚀 중급자 (3-7년차)  
```
프롤로그 → 4-8장 → 9-12장 → 케이스 스터디
```
**포커스**: 고급 분석, 리더십, 크로스 펑셔널 협업

#### 🎯 고급자 (7년+ 리더/컨설턴트)
```
프롤로그 → 9-15장 → 전략 프레임워크 → 전체 리뷰
```
**포커스**: 전략 설계, 컨설팅 방법론, 조직 변화 관리
